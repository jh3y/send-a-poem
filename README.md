# Send a Poem 🛩
A small app for grabbing a random poem from `PoetryDB` and composing an email that you can send via a `mailto:` link 👍

Built for the `#CodePenChallenge`.

Built with `parcel`, `react`, `PoetryDB` and `greensock`.
